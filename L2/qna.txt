1. [77, 78, 79]
Интервал для слайса не включает индекс справа - a[1:4] == a[1, 2, 3]

2. 2, 1
defer-функция изменит только именнованные параметры после того, 
как окружающая функция достигает return

3. nil, false
Две переменные, удовлетворяющие интерфейсу равны только если равны
их динамические типы и значения. В данном случае типы разные - 
os.PathError и nil

4. Числа 0...9, затем ошибка
Т.к. канал не был закрыт, функция main() остановится в цикле for range ch,
ожидая дальнейшего поступления данных в канал

5. error
После присвоения err = test() динамический тип err равен *customError
Интерфейс равен nil только если динамический тип равен nil

6. [3, 2, 3]
Слайс передается в функцию по значению, но внутри содержит указатель на массив данных
Обращения по индексу модифицируют изначальные значения массива
Если при попытке append будет превышена вместимость изначального массива, 
то он будет увеличен и реаллоцирован, изначальный указатель будет потерян,
дальнейшие изменения будут касаться уже нового массива

7. Как минимум 1, 3, 5, 7 или 2, 4, 6, 8, возможно с промежуточными числами,
из списка переданных в asChan, затем нули. 
В merge() отсутствует условие выхода из цикла for, поэтому после закрытия a или b
канал c будет бесконечно отдавать нули
